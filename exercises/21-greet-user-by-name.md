# Exercise 21: Greet the User by Name

**Level:** Intermediate
**Skills practiced:** Getting user input and displaying dynamic content

---

## Your Prompt

Copy and paste this into Claude:

> Add a new card below the date card with a text input box where someone can type their name, and a "Say Hello" button. When clicked, the page should display a greeting like "Hello, [Name]! Welcome to the page." in large friendly text.

---

## What to look for when done

- A text input and button appear in the new card
- Typing a name and clicking the button shows the greeting
- The greeting updates every time you click with a different name
