# Exercise 29: Build a Notes App That Saves

**Level:** Advanced
**Skills practiced:** localStorage (browser storage), save/load/clear operations

---

## Your Prompt

Copy and paste this into Claude:

> Add a "My Notes" card with a large text area for writing notes, a "Save Note" button, and a "Clear Note" button. When I click "Save Note", the note should be saved in the browser so it is still there even after I close and reopen the page. When I click "Clear Note", it should delete the saved note and empty the text area. Show a small message like "Saved!" when saving.

---

## What to look for when done

- Typing a note and clicking Save persists it
- Closing and reopening the browser still shows the note
- Clear Note removes it completely
- A "Saved!" message appears briefly after saving
