## Your Prompt

Copy and paste this into Claude:

> Add a "Countdown Timer" card. It should have a number input where I can enter how many seconds to count down from, a "Start" button to begin the countdown, and the remaining seconds should be displayed in large text, counting down to zero. When it reaches zero, show a "Time's up!" message and play a short visual flash effect.

---

## What to look for when done

- Entering a number and clicking Start begins the countdown
- The number visibly counts down every second
- It stops at zero and shows "Time's up!"
- There is a visual effect when time runs out
- The Start button is disabled while counting (so you can't start twice)
