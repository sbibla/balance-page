# Demo Project — Claude Context

## What This Project Is

A simple practice webpage for learners who are new to Claude Code. It has three files:

- `index.html` — the structure and content of the webpage
- `style.css` — the visual styling (colours, fonts, layout)
- `app.js` — the interactive behaviour (what happens when you click the button)

## Who Is Working On This

A non-developer learning to use Claude Code for the first time. They may not know HTML, CSS, or JavaScript.

## How Claude Should Help Here

- Explain every change in plain English before and after making it
- When editing `style.css`, describe what each property controls (e.g., "background-color sets the page background")
- When editing `app.js`, explain JavaScript in analogies (e.g., "a variable is like a labelled box that stores a value")
- Suggest the next small thing to try after completing a task
- Keep changes small and understandable — don't refactor everything at once

## Project Rules

- Do not add external libraries or frameworks (no React, Bootstrap, jQuery, etc.)
- Keep all code in these three files — no new files unless the learner specifically asks
- Always open the browser to see changes: `index.html` can be opened directly by double-clicking it

## Suggested Practice Sequence (for the learner)

1. Change the header background colour in `style.css`
2. Update the welcome text in `index.html`
3. Add a new task to the "Practice Tasks" list
4. Modify the button messages in `app.js`
5. Add a new section (card) to the page
6. Make the button change colour on click
7. Add the current date/time somewhere on the page
8. Add a second button that does something different
